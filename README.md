## Installation

1. `git clone https://github.com/ntzm/trucktrack.git`
1. `cd trucktrack`
1. `cp .env.example .env`
1. Edit `.env` to your liking
1. `php artisan key:generate`
1. `composer install`
1. `npm install`
1. `npm run dev`
